# nodejs_stock_loader
* sotck loader for node js implement
* load all the stock's code and name;



example installation:
* npm install xpath
* npm install xmldom
* npm install gbk
* npm install cheerio
* npm install colors
* npm install node-adodb

npm install mocha
npm test
